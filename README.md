# Network Topology Construction Project

## Description
Designed and configured a open-source web application penetration testing tool by integrating different types of cyberspace search engines and active reconnaissance tools like nmap and zmap and also integrating the metasploit framework.
Implemented security measures to ensure efficient and secure data communication, followed by rigorous testing for functionality, connectivity, and performance within the simulated environment.
